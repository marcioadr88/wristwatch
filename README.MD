# Requirements

In order to run the project be sure you have swiftlint installed at system level. Check the installation instructions at their [website](https://github.com/realm/SwiftLint).

# Dependencies

The project is using Swift Package Manager to manage the library dependencies, once you open the project with XCode it will automatically download the dependencies for you.

## Libraries

- **RxSwift**: Reactive programming, observables, binding.
- **Realm**: Local database
- **Kingfisher**: Asynchronous image downloading and caching
